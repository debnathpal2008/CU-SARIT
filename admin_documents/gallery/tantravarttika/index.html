Tantravārttika

This sample is the first volume of the Tantravārttika (covering pādas 1.2 to 2.1). It has “standard” features: structural and block-level markup, selected markup of inline elements (e.g., bold text), and notes. The HTML files are formatting using the standard stylesheets (used also for the Kāvyādarśa).
