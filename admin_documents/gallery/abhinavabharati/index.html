Abhinavabhāratī

This sample is chapter 4 of Abhinavagupta’s Abhinavabhāratī, together with the text that it comments upon, chapter 4 of Bharata’s Nāṭyaśāstra.

This sample tests a new system: instead of preparing the HTML files from the XSL stylesheets beforehand, the XSL stylesheet is loaded and called from within the webpage, and the HTML files are generated on the client side. This is my preferred way of doing XML-to-HTML transformations, which will be useful for selecting portions of XML to display on-demand. Both files are loaded in windows that can be moved and resized: this seems to be a useful format for reading texts together with their commentaries.

Both texts include variant readings (represented as “footnotes” that display the readings on mouseover) and the Abhinavabhāratī contains pratīkas that point to the corresponding verse in the text of the Nāṭyaśāstra.

Only a devanāgarī version of this sample is available at the moment.
